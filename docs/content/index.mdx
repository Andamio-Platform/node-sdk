Let's discover **Andamio by using our SDK**.

## Getting Started

Get started by **installing Andamio SDK**.

### Using `npm`

```bash
npm install @andamiojs/sdk
```

#### Initiation

```typescript
const sdk = new AndamioSDK(<base-url>, <network>)
```

##### Using demeter

```typescript
const sdk = new AndamioSDK("https://preprod.utxorpc-v0.demeter.run:443", "Preprod", "dmtr_utxorpc...")
```

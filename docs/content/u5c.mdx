# Introduction to UTxO RPC

[UTxO RPC](https://utxorpc.org) is a modern, high-performance RPC framework specifically designed for UTxO-based blockchains like. It provides developers with tools to interact with blockchain networks efficiently through standardized APIs.

The Andamio SDK is built on the UTxO RPC standard, leveraging this powerful foundation to provide a seamless development experience across UTxO-based blockchains.

## Key Features

- **gRPC-based**: Uses Protocol Buffers for efficient, type-safe communication
- **Streaming Support**: Real-time updates for blockchain events
- **Developer-Friendly**: Clean, consistent APIs designed for modern development
- **Blockchain Agnostic**: Built to work across multiple UTxO-based cryptocurrencies
